shell 中的 case 語句 (switch)

語法介紹

有點像是其他語言的switch

  • 每個pattern後面接 ) ， 只要符合該pattern，就會執行後面對應的語句

  • ;; 相當於 break

  • * 相當於 default

  • expression 可以是一個變數，或是某個指令執行的結果

  • pattern 可以是數字、字串，也可以使用正則表達式
